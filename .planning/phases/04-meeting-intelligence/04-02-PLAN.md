    **4. Create `src/meeting_transcriber/core/meeting_detector.py`:**
    Module docstring: `"""회의 앱 감지 워커 — 백그라운드 폴링."""`

    Class `MeetingDetectorWorker(QThread)`:
    - Signals:
      - `meeting_detected = pyqtSignal(str, str)` — (app_name, suggested_template_key)
    - `__init__(self, *, parent=None)`:
      - `self._running = False`
      - `self._snoozed: set[str] = set()` — snoozed bundle IDs
      - `self._last_notify_time: float = 0.0` — for global cooldown
      - `self._is_recording = False` — flag set by UI to suppress during recording
    - `start_detection(self)`: sets `self._running = True`, calls `self.start()`
    - `stop_detection(self)`: sets `self._running = False`, calls `self.wait()`
    - `set_recording(self, recording: bool)`: sets `self._is_recording = recording`
    - `snooze(self, bundle_id: str)`: adds to `self._snoozed`
    - `run(self)`:
      ```python
      while self._running:
          try:
              self._poll_once()
          except Exception:
              pass  # Silent failure per UI-SPEC error states
          self.msleep(DETECTION_POLL_INTERVAL_MS)
      ```
    - `_poll_once(self)`:
      - Lazy import: `from AppKit import NSWorkspace`
      - Get running apps via `NSWorkspace.sharedWorkspace().runningApplications()`
      - Build set of running bundle IDs
      - Clear snoozed bundle IDs that are no longer running: `self._snoozed -= (self._snoozed - running_bids)`
      - If `self._is_recording`: return early (per D-16 — no prompts during active recording)
      - Check global cooldown: `time.time() - self._last_notify_time < DETECTION_COOLDOWN_SECONDS` → skip
      - For each running app whose bundle ID is in KNOWN_CONFERENCING_APPS and NOT in self._snoozed:
        - If bundle ID is CHROME_BUNDLE_ID: call `self._check_audio_activity()` — skip if False
        - Emit `self.meeting_detected.emit(app.localizedName(), KNOWN_CONFERENCING_APPS[bid])`
        - Update `self._last_notify_time = time.time()`
        - Return after first detection (one notification per poll cycle)
    - `_check_audio_activity(self) -> bool`:
      ```python
      try:
          import sounddevice as sd
          import numpy as np
          # Record 0.5 seconds from default input to check activity
          recording = sd.rec(int(0.5 * 16000), samplerate=16000, channels=1, dtype="float32")
          sd.wait()
          rms = float(np.sqrt(np.mean(recording ** 2)))
          return rms > 0.005  # Low threshold — just checking if mic is active
      except Exception:
          return False  # If audio check fails, don't trigger
      ```
