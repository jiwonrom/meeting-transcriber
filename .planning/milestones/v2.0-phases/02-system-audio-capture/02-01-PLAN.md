"""시스템 오디오 캡처 — BlackHole 감지, Aggregate Device 생성/해제."""

**Functions to implement:**

1. `detect_blackhole() -> int | None` — Iterate `sd.query_devices()`, find first device whose `name.lower()` matches any entry in `BLACKHOLE_DEVICE_NAMES` and has `max_input_channels > 0`. Return index or None. Wrap in try/except for `sd.PortAudioError` returning None. Handle both dict (single device) and list (multiple devices) return from `query_devices()`.

2. `is_blackhole_installed() -> bool` — Returns `detect_blackhole() is not None`.

3. `get_device_uid(device_index: int) -> str` — Use `CoreAudio.AudioObjectGetPropertyData` with selector `kAudioDevicePropertyDeviceUID`, scope `kAudioObjectPropertyScopeGlobal`, element `kAudioObjectPropertyElementMain` (value 0). The property address is constructed as `CoreAudio.AudioObjectPropertyAddress(kAudioDevicePropertyDeviceUID, kAudioObjectPropertyScopeGlobal, 0)`. Returns the UID as a string. Raises `SystemAudioError` if the call fails. Import `CoreAudio` at function level (lazy import) to avoid import-time failure on systems without pyobjc.

4. `create_aggregate_device(mic_uid: str, blackhole_uid: str) -> int` — Construct the aggregate device description dict per RESEARCH.md Pattern 2:
   - `kAudioAggregateDeviceNameKey`: `AGGREGATE_DEVICE_NAME`
   - `kAudioAggregateDeviceUIDKey`: `AGGREGATE_DEVICE_UID`
   - `kAudioAggregateDeviceIsPrivateKey`: 1 (private, process-scoped)
   - `kAudioAggregateDeviceSubDeviceListKey`: list of dicts with `kAudioSubDeviceUIDKey` for both mic_uid and blackhole_uid
   - `kAudioAggregateDeviceMasterSubDeviceKey`: blackhole_uid (BlackHole as clock master per Pitfall 2)
   Call `CoreAudio.AudioHardwareCreateAggregateDevice(description, None)`. Returns `(err, device_id)` tuple. If err != 0, raise `SystemAudioError(f"Failed to create Aggregate Device (OSStatus {err})")`. Return device_id. Lazy import CoreAudio.

5. `destroy_aggregate_device(device_id: int) -> None` — Call `CoreAudio.AudioHardwareDestroyAggregateDevice(device_id)`. Log error but don't raise on failure (cleanup should not crash). Lazy import CoreAudio.

6. `resolve_device_by_uid(uid: str) -> int | None` — Iterate `sd.query_devices()` and for each input device, call `get_device_uid(i)` in try/except. Return index if UID matches. Return None if not found. This is slow but only called once at recording start.
