<objective>
Fix MetadataIndex language field mismatch and wire index updates through update_transcript_speakers.

Purpose: CMA-03 requires a working searchable metadata index. Two bugs prevent this: (1) MetadataIndex reads `meta["language"]` (singular) but v2.0 transcripts store `meta["languages"]` (plural list), causing language data loss in the index; (2) `update_transcript_speakers()` does not propagate MetadataIndex updates, so diarization results never update the index.

Output: Fixed metadata_index.py, updated transcript_store.py with optional index parameter, updated main_window.py to pass index, comprehensive tests.
</objective>
